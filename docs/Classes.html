  • LocationEntry

    The LocationEntry class is one level removed from the RawCoordinates; it stores a collection of RawCoordinates as well as a central latitude and longitude for the given set of coordinates. It also manages a start and end time, as well as a movement type for the set of entries. Notably, LocationEntry can represent both STATIONARY locations (places like restaurants, gyms, etc.) a nd legs of commute (in which case the latitudes and longitudes will be -1.0, -1.0) and movement_type will either be AUTOMATIVE, CYCLING, or WALKING.
